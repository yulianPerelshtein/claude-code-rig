# templates/

Per-project starter files. Copy into a new project and fill in; nothing here is
auto-loaded.

| File | Use |
|---|---|
| `project-CLAUDE.md.tmpl` | A new project's Layer-2 `CLAUDE.md`. |
| `project-AGENTS.md.tmpl` | A new project's `AGENTS.md` (agent hygiene rules). |
| `python-flake8-79char.md` | Drop-in Python lint/format conventions for a 79-char project. |
| `repo-onboarding.md` | First-day orientation notes for a repo. |
| `repo-audit/new-repo-audit-checklist.md` | Structured audit of an unfamiliar repo. |
