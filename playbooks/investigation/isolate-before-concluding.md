# Isolate before concluding

When a symptom has two (or more) plausible causes, isolate each **independently**
before concluding which is responsible.

- A combined test (A and B both active) masks which one matters — and can show
  a "fix" that's really the other variable.
- Toggle one variable at a time; record the result of each isolated run.
- Only attribute the cause once a single-variable test confirms it.
